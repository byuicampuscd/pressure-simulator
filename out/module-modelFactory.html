makeMeasureModel(measureBound, precision)

This is the method to create and return a MeasureModel
Parameters:
Name Type Description
measureBound number | Array.<number> Either a number specifying the max of the measure, or an array specifying the [min,max] of the measure. The default min is 0.
precision int The desired number of decimals places to round to when the number is displayed. This number does not affect the underlying model.
